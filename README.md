# PRISM RIFT VANGUARD

Three.js ベースで制作された、キャンペーン進行型の 3D シューティングゲームです。  
各ミッションを突破しながらクリスタルを集め、任務間インターバルのハンガーショップで機体を強化し、最終決戦の制覇を目指します。

---

## 概要

**PRISM RIFT VANGUARD** は、

- ミッション制 3D シューティング
- ミッション間のショップ強化によって機体性能を拡張する進行型構成
- 通常ショットとクールタイム制プラズマショットを軸にした戦闘

を備えた、ブラウザ実行型のキャンペーンシューターです。

現在の実装では、タイトル画面から **チュートリアル** と **キャンペーン開始** を個別に選択できます。  
本編は複数ミッションで構成され、各空域ごとに敵構成、演出、地形テーマ、環境ギミック、ボスが切り替わります。

---

## 主な特徴

- **キャンペーン進行型**
  - チュートリアルの後、本編キャンペーンを順番に攻略します。
  - 各ミッションは複数 WAVE とボス戦、またはボス直行構成で進行します。

- **ハンガーショップによる成長要素**
  - ミッション間インターバルでクリスタルを消費し、機体性能を強化できます。
  - メインショット系とプラズマ系に分かれた強化項目があります。

- **記録確認画面**
  - ミッションごとのハイスコア、最速タイム、獲得勲章を確認できます。
  - キャンペーン最終結果も保存されます。

- **多言語対応**
  - 日本語 / 英語 / 中国語に対応しています。

- **オプション**
  - オプション画面はタイトル画面だけでなく、ポーズ中・インターバル中からも開けます。

## 起動方法

### 1. ローカルサーバーを起動

このプロジェクトのルートディレクトリで、たとえば次のようにサーバーを立ち上げます。

```bash
python -m http.server 8000
```

### 2. ブラウザで開く

```text
http://localhost:8000/
```

`index.html` が読み込まれ、タイトル画面が表示されます。

## 操作方法

### 基本操作

- **W / A / S / D** : 移動
- **マウス移動** : 視点操作
- **左クリック** : 通常ショット
- **右クリック** : プラズマショット
- **Esc** : ポインタロック解除 / ポーズ移行

### プレイ中の挙動

- ゲーム中はポインタロックでマウス視点になります。
- プレイ中に `Esc` でポインタロックを解除すると、一時停止画面へ移行します。
- 再開時は再度ポインタロックを取得します。
- 旧 README にあった **Shift による高速移動は現在の実装にはありません**。

## ゲームの流れ

1. タイトル画面から **チュートリアル** または **キャンペーン開始** を選択
2. ミッションを攻略してクリスタルを回収
3. 任務間インターバルでショップ強化
4. 次ミッションへ出撃
5. 最終ミッション突破でキャンペーンクリア

### 戦闘の基本

- **通常ショット**
  - 主力武装です。
  - 強化により威力、連射性能、貫通、多方向射撃、ホーミング補助を伸ばせます。

- **プラズマショット**
  - クールタイム制の高威力攻撃です。
  - 強化により直撃 / 爆風ダメージ、爆風半径、クールタイム、同時発射数を変化させられます。

- **クリスタル**
  - 敵撃破や進行報酬で獲得します。
  - ハンガーショップでの強化に使用します。

## ミッション一覧

### チュートリアル
- **飛行訓練** (`FLIGHT TRAINING`)
  - 基本移動、通常ショット、プラズマ、戦利品回収を短時間で確認する訓練空域です。

### キャンペーン
1. **グラスデザート** (`GLASS DESERT`)
2. **ルミナスマーシュ** (`BIOLUMEN SWAMP`)
3. **エンバーフォージ** (`EMBER FORGE`)
4. **フロストヴェール** (`FROST VEIL`)
5. **ミラーシタデル** (`MIRROR CITADEL`)
6. **アストラルリーフ** (`ASTRAL REEF`)
7. **ヴォイドクラウン** (`VOID CROWN`)

各ミッションには固有の敵構成、視覚演出、地形テーマ、ボス、環境ギミックが設定されています。

## 強化要素

ショップでは主に以下を強化できます。

- 最大ライフ
- 装甲
- クリスタル吸引
- 通常ショット威力
- 通常ショット連射性能
- 貫通弾
- 多方向射撃
- ホーミング補助
- プラズマ威力
- プラズマ爆風半径
- プラズマ冷却短縮
- プラズマ多方向化

強化は段階式で、項目ごとに最大レベルが設定されています。

> 同時発射数を増やす強化には、クールタイム増加のトレードオフがあります。

## タイトル画面から使える機能

### 1. チュートリアル
基本操作の確認用ミッションを開始します。

### 2. キャンペーン開始
本編キャンペーンを最初の実戦ミッションから開始します。

### 3. レコード
保存された記録を確認できます。

- ミッションごとのハイスコア
- ミッションごとの最速タイム
- ミッションごとの獲得勲章
- キャンペーン最終結果
- 保存データ削除

### 4. オプション
各種設定を変更できます。

### 5. クレジット
使用プログラム、BGM、SE のクレジットを確認できます。

## オプション項目

現在の実装で変更できる主な項目は以下の通りです。

### システム
- 言語

### オーディオ
- サウンドテスト

### 操作

- 照準サイズ
- 被弾方向表示

### HUD
- HUD 不透明度
- ミニマップ表示
- ミニマップ拡大率
- 敵マーカー表示
- 高コントラスト表示

また、オプション画面から **規定値に戻す** 操作も行えます。

## デバッグ機能

デバッグ画面は通常プレイでは非表示です。  
URL に `?debug=1` を付けて起動すると有効になります。

例:

```text
http://localhost:8000/?debug=1
```

デバッグ画面では、たとえば次のような機能が利用できます。

- タイトルから開始するミッションの変更
- GODMODE 切り替え
- BOSS MODE 切り替え
- パフォーマンスレポート確認

実行中のデバッグ操作として、画面内敵の一掃などの補助導線も組み込まれています。

> デバッグ機能は開発・確認用です。通常プレイ用の導線とは分離されています。

## 保存データ

本作はブラウザの **localStorage** を使って、オプション設定とプレイ記録を保存します。

### 保存される主な内容

- オプション設定
  - 言語
  - サウンドテスト選択曲
  - BGM / SE 音量
  - マウス感度
  - Y 軸反転
  - 描画品質
  - FOV
  - エフェクト強度
  - HUD / 照準 / ミニマップ関連設定

- プレイ記録
  - ミッションごとのハイスコア
  - ミッションごとの最速タイム
  - ミッションごとの獲得勲章
  - キャンペーン最終結果

### データ削除

タイトル画面の **レコード** から削除できます。


### 構成の目安

- `src/data/` : ミッション、敵、バランス、音声定義などのデータ
- `src/player/` : プレイヤー移動、武装、被弾、演出
- `src/enemies/` : 通常敵、ボス、敵 AI
- `src/progression/` : ミッション進行、インターバル、実績 / 勲章、強化
- `src/ui/` : タイトル、レコード、図鑑、オプション、クレジット、HUD
- `src/storage/` : localStorage ベースの設定 / 記録保存
- `src/world/` : 地形、空域テーマ、環境オブジェクト
- `src/render/` : Three.js の描画制御
- `src/debug/` : デバッグ導線と計測補助
